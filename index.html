Principais Projetos

Sistema de Pesquisa de Satisfação e Plano de Ação

Desenvolvimento de um sistema de pesquisa de satisfação para avaliação de áreas e serviços, com um mecanismo automatizado que gera planos de ação para áreas com baixa performance. As áreas são notificadas automaticamente e acompanham a execução dos planos dentro do próprio sistema. A solução foi implementada utilizando ASP.NET Core e SQL Server, integrando módulos de relatórios e gráficos para análise.

Sistema de Integração de Gestão

Desenvolvimento de um sistema de integração para unificar e sincronizar dados de diferentes sistemas de gestão, como Senior, Qualiex, Totvs e MKData. A integração foi realizada utilizando Pentaho, conectando APIs RESTful e diferentes bancos de dados, incluindo Progress, SQL Server e Oracle. O sistema garante o fluxo contínuo de informações, consistência de dados e redução de retrabalhos nos processos de gestão. O projeto incluiu monitoramento de logs e verificações automáticas para assegurar a confiabilidade das transações entre os sistemas.

Sistema de Controle de Atendimento e Vendas (SCAVE)

Desenvolvimento do SCAVE, acessado por meio de aplicação web, utilizando Java e JavaScript. O sistema segue o padrão Model-View-Controller (MVC) e armazena dados em PostgreSQL, integrando APIs REST para um gerenciamento eficiente das informações.

Ô Moto

Aplicativo desenvolvido para otimizar a experiência do usuário no transporte coletivo, proporcionando informações em tempo real sobre horários, trajeto e valores de passagens. O foco é reduzir atrasos e melhorar a acessibilidade ao transporte público.

Depois da Escola

Sistema educacional que visa facilitar o acesso a conteúdos escolares, oferecendo uma plataforma online para estudantes. O projeto utiliza PHP, MySQL e JavaScript, promovendo uma aprendizagem acessível e inovadora.

Smart Mirror

Desenvolvimento de um espelho inteligente que exibe informações úteis, permitindo multitarefas de forma “hands-free”. O projeto utiliza tecnologia de IoT e é uma combinação de software e hardware.
